----------------------------------------------------------------------
PRICE MOVEMENT SIMULATION:
----------------------------------------------------------------------

Function: _simulate_price_movement(current_price, volatility)
File: services/investment_service.py (Line 57-73)

Formula:
    1. Direction decide hota hai (60% chance up, 40% chance down)
    2. Volatility ke hisab se max change calculate hota hai
    3. Random change apply hota hai
    4. New price = current_price * (1 + random_change)

Example (BITCOIN):
    - Current Price: ₹45,00,000
    - Volatility: 8%
    - Max Change: 8% = 0.08
    - Random Change: 0.05 (randomly between 0 and 0.08)
    - New Price: ₹45,00,000 * (1 + 0.05) = ₹47,25,000

    YA
    
    - Current Price: ₹45,00,000
    - Volatility: 8%
    - Direction: -1 (down)
    - Random Change: -0.03
    - New Price: ₹45,00,000 * (1 - 0.03) = ₹43,65,000

================================================================================
                    2. USER INVESTMENT VALUE UPDATE
================================================================================

Jab user koi asset kharidta (buy) ya bechta (sell) hai, toh:

BUY (KHARIDNA):
---------------
1. User wallet se amount deduct hota hai
2. User ke investment record mein units add hote hain
3. Average buy price calculate hota hai

Code Location: services/investment_service.py (Line 79-203)

Example (BITCOIN):
    - User ₹1,00,000 invest karta hai
    - Current Price: ₹45,00,000/unit
    - Units Milenge: 1,00,000 / 45,00,000 = 0.0222 units
    
    - Agar user pehle se 0.01 BTC owned karta hai
    - Naye total: 0.01 + 0.0222 = 0.0322 BTC
    - Total invested: ₹45,000 + ₹1,00,000 = ₹1,45,000
    - Average Price: ₹1,45,000 / 0.0322 = ₹45,03,105/unit

SELL (BECHNA):
--------------
1. User ke units kam ho jate hain
2. Wallet mein sale amount add hota hai
3. Profit/Loss calculate hota hai

Code Location: services/investment_service.py (Line 209-323)

Example (BITCOIN):
    - User 0.01 BTC bechta hai
    - Current Price: ₹50,00,000/unit
    - Sale Amount: 0.01 * 50,00,000 = ₹50,000
    
    - Buy Price: ₹45,00,000/unit
    - Cost Basis: 0.01 * 45,00,000 = ₹45,000
    - Profit: ₹50,000 - ₹45,000 = ₹5,000

================================================================================
                    3. PORTFOLIO VALUE CALCULATION
================================================================================

Function: get_portfolio()
File: services/investment_service.py (Line 329-385)

Har holding ke liye yeh calculate hota hai:

    Current Value = units_owned * current_market_price
    Profit/Loss = Current Value - Invested Amount
    Profit/Loss % = (Profit/Loss / Invested Amount) * 100

Example (BITCOIN):
    Units Owned: 0.05 BTC
    Buy Price: ₹40,00,000/unit
    Invested Amount: ₹2,00,000
    Current Market Price: ₹50,00,000/unit
    
    Current Value: 0.05 * ₹50,00,000 = ₹2,50,000
    Profit/Loss: ₹2,50,000 - ₹2,00,000 = ₹50,000
    Profit/Loss %: (50000/200000) * 100 = 25%

Total Portfolio:
    Total Invested = Sum of all invested amounts
    Total Current Value = Sum of all current values
    Total P/L = Total Current Value - Total Invested
